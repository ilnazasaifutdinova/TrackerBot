# Tracker Discord Bot

## Установка
```bash
git clone https://github.com/ilnazasaifutdinova/TrackerBot.git
cd TrackerBot
pip install -r requirements.txt
```

Создайте файл `.env`:
```env
DISCORD_TOKEN=ваш_токен_бота
CHANNEL_ID=id_канала_для_уведомлений
MANGA_URL=https://ваша-манга
```

Запустите:
```bash
python bot.py
```
